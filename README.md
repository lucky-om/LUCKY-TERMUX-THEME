## INSTALL WITH TERMUX :

```
apt update
```
```
apt upgrade -y
```
```
pkg install git -y
```
```
git clone https://github.com/lucky-om/LUCKYX
```
```
cd LUCKYX/
```
```
chmod +x *
```
```
bash install.sh
```

### RUN :

```
exit
```

### AT ONCE :

```
apt update && apt upgrade -y ; pkg install git -y ; git clone https://github.com/lucky-om/LUCKYX; cd LUCKYX/ ; chmod +x * ; bash install.sh
```
